# Velo

We're still unsure whats its about, but lets see how it goes

## Velo-frontend

Start up velo frontend with

```bash
cd velo-frontend
npm install
npm run dev
```

To build project

```bash
npm run build
```

To view preview of build

```bash
npm run preview

```

## Velo-backend

STart up velo backend with

```bash
cd velo-backend
pip install -r requirements.txt
```
